# Task 009 Plan: Codec/gRPC Generation Chain Cleanup

## Status

Ready for implementation.

## Starting Point

Current repository state was committed before this plan:

```text
e076fe6 codgent: complete fanout friend group snapshot
```

The previous baseline is verified:

- ODB-enabled: 14/14 tests passed.
- No-ODB: 2/2 tests passed.
- Codec explicit enable: `im_codec_service` builds when gRPC is available.

## Goal

Make protobuf and codec gRPC generation deterministic from a single canonical
source tree, `common/proto`, before standalone Push Service work begins.

## Current Findings

- `common/proto` is the canonical proto source directory.
- Root `generate_proto` generates only C++ protobuf `.pb.cc/.pb.h` files.
- gRPC stubs are not generated by the root CMake target.
- `services/codec` contains duplicated generated `base.pb.*`,
  `codec_service.pb.*`, and `codec_service.grpc.pb.*`.
- `services/codec/CMakeLists.txt` is unused.
- `services/CMakeLists.txt` creates `im_codec_service` directly when
  `MYCHAT_BUILD_CODEC_SERVICE=ON`.
- `im::proto` does not compile `common/proto/codec_service.pb.cc`.
- `services/codec/main.cpp` is only a placeholder.
- CMake selects vcpkg `protoc-29.3.0`; shell `protoc` is 34.1 and must not be
  used for checked-in generated files.

## Required Changes

1. Add deterministic gRPC plugin discovery in CMake.
2. Add or update generation targets so all active `.pb.*` and `.grpc.pb.*`
   outputs are generated from `common/proto`.
3. Choose and document the generated output location before editing generated
   files. Preferred short-term choice: keep generated outputs in `common/proto`.
4. Update `im::proto` and `im_codec_service` so `codec_service.pb.*` and
   `codec_service.grpc.pb.*` come from the canonical output path.
5. Stop compiling duplicated `services/codec/base.pb.*`.
6. Keep `MYCHAT_BUILD_CODEC_SERVICE=OFF` default behavior unchanged.
7. Add a verification command for `MYCHAT_BUILD_CODEC_SERVICE=ON`.

- Default builds remain unchanged when codec service is OFF.
- ODB and no-ODB regression baselines still pass.
