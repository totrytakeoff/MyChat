# 历史文档归档

本目录保存 MyChat 开发过程中的阶段日志、协作记录和旧报告。

## 子目录

- `devlog/`：按阶段记录的开发日志。
- `codgent/`：早期 codgent 多 agent 实验输出。
- `agent_context/`：早期 agent/codgent 协作上下文、任务记录、运行日志。
- `root_old/`：原 `docs` 顶层旧报告。

## 使用说明

这些文档用于追溯项目演进，不作为当前项目现状的权威来源。文档中可能包含旧路径、旧设计、旧测试状态或已被后续实现替代的计划。

当前状态请优先阅读：

- `docs/project/roadmap/current_progress.md`
- `docs/project/roadmap/service_mvp_roadmap.md`
- `docs/project/architecture/mvp_architecture.md`
- `docs/final_sum_docs/`
